An ArcAPI for bots running in NodeJS. <br/>

## 🤔 Dependences
- node >= `v12.16.0`
- npm >= `v6.14.0`
- sqlite3 >= `v4.1.1`

## 🌎 Clone
> $ git clone https://git.moezsoft.com:4333/konata-dev/botarcapi  
> $ git checkout master

## 🎉 Run directly
> $ npm run start
