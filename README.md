# BotArcapi

一個運行在 `CloudflareCDN` 的 Arcaea API <br>

**僅限内部使用**

---

#### 🤔 先行條件
- npm >= `6.13.7`
- webpack >= `4.41.5`
- webpack-cli >= `3.3.10`

#### 🌎 克隆倉庫
> $ git clone https://git.moezsoft.com:4333/arcaeascorebot-developer/botarcapi.git  
> $ git checkout master

#### 🎉 修改版本號
> $ nano webpack.config.js

修改 `BOT_API_VERSION` 版本 並保存

#### 🚀 開始打包
> $ npm run build

#### 🤦‍♀️ 許可證
根據 `WTFPL` 許可證授權分發 <br>
想怎麽用就怎麽用 草 (但僅限内部使用
